Qwiz

Qwiz was born during a hackathon and is a SaaS Quiz webapp. It uses Django and PostgreSQL for the backend, connected to an Angular2 frontend using Redis message queues. Check out our team GitHub.

GitAssist

GitAssist was created for a hackathon and provides a Google Assistant skill to link to a GitHub account, adding voice commands to query issues and manage pull requests. I was in charge of the natural language processing, using Dialogflow, and my work can be found on GitHub.
